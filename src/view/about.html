What is Simple Hashtag Manager?

This tool provides the following functionality:

  • Add hashtags, organized by favorite and categories
  • Get a number of hashtags for a category (favorite hashtags are with priority with the random option)
  • Copy those hashtags to clipboard for ease of use and edit them
  • You can export all of your categories with hashtags as a JSON file and you can import them back

Simple Hashtag Manager uses your browser's LocalStorage to store your hashtags data which means that if you clear your browser's data you will lose all of your hashtags in this app!
To be safe, make sure to use the 'export' button in the 'Home' page to export your data as a file that you can keep and later import in Simple Hashtag Manager.
